A simple and lightweight plugin to customize the WordPress login page with your logo, background color, and button colors.

== Description ==

Personalized Login allows you to easily customize your WordPress login page. Upload your own logo, set a custom background color, and choose a button color to create a fully branded login experience. No coding required!

== Installation ==

1. Upload the `personalized-login` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Settings > Login Branding** to customize the login page.

== Frequently Asked Questions ==

= How do I upload my custom logo? =
You can upload your logo directly from the plugin settings page by clicking on the "Upload Logo" button.

= What file formats are supported for the logo? =
You can upload JPEG, PNG, or GIF images as your custom logo.

= Can I revert back to the default WordPress login page? =
Yes, you can disable the plugin at any time to revert to the default login page.
